# 🌟 Aseni Thennakoon - Personal Portfolio

🚀 **Live Website:** [https://portfolio-theta-amber-65.vercel.app/](https://portfolio-theta-amber-65.vercel.app/)

Welcome to the repository for my personal portfolio! This project is a modern, highly interactive web application designed to showcase my skills, projects, and professional journey as a Software Engineer.

## ✨ Key Features

- **Interactive 3D Hero:** A stunning, interactive 3D Spline scene on desktop, gracefully degrading to a clean, animated CSS design for smaller screens.
- **Dynamic Theming:** Seamless Dark and Light modes featuring beautiful glassmorphism panels, glowing gradients, and customized typography.
- **Motion & Interactions:** 
  - Dynamic 3D tilt-effects on project cards based on mouse movement.
  - Infinite smooth-scrolling tech stack marquees.
  - Typewriter text effects in the hero section.
  - Intersection Observers for elegant scroll-reveal animations.
- **Project Showcase:** A clean, paginated grid displaying my latest full-stack, mobile, and frontend projects with detailed contributions.
- **Functional Contact Form:** Fully integrated with EmailJS, allowing visitors to send messages directly to my inbox with instant visual feedback.
- **Responsive Design:** A mobile-first approach ensuring the portfolio looks perfect on phones, tablets, and large desktop monitors.

## 🛠️ Tech Stack & Tools

This portfolio was built using modern web development tools:

### Core Technology
- **[React 18](https://react.dev/)** - Component-based UI development.
- **[Vite](https://vitejs.dev/)** - Next-generation front-end tooling for lightning-fast builds and hot module replacement.

### Styling & UI
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework used for rapid responsive styling.
- **Vanilla CSS** - Extensive custom CSS (`index.css`) for advanced animations, gradients, scrollbars, and structural overrides.
- **[Lucide React](https://lucide.dev/)** - Beautiful, consistent SVG icons used throughout the application.

### Integrations & Libraries
- **[@splinetool/react-spline](https://spline.design/)** - Rendering the interactive 3D models.
- **[EmailJS](https://www.emailjs.com/)** - Handling contact form submissions without requiring a dedicated backend server.
- **[React Simple Typewriter](https://www.npmjs.com/package/react-simple-typewriter)** - Easy typing animation effects.

## 🚀 Getting Started

If you want to run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Thennakoon03/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your EmailJS credentials:
   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173` to see the app running!
